# Sound Files

Place your audio files in the appropriate subdirectory:

```
Sounds/
├── pain/       — "Ow", "Hey", protest sounds
├── sexy/       — Escalating moans/reactions
├── halo/       — Game sound effects
└── custom/     — (Not used directly — user picks their own folder)
```

## Supported Formats

- MP3 (.mp3)
- WAV (.wav)
- AAC (.aac, .m4a)
- AIFF (.aif, .aiff)
- Core Audio (.caf)

## Placeholder Files

The app ships without bundled sounds to avoid copyright issues.
Add your own MP3/WAV files to each folder. The app will load all
audio files it finds in the active mode's directory.

## Suggested Sources for Free Sounds

- freesound.org (CC0 and Creative Commons sounds)
- pixabay.com/sound-effects (royalty-free)
- Record your own reactions!

## Naming Convention

Name files descriptively — they're loaded alphabetically but played
in shuffled order. Examples:

```
pain/ow-1.mp3
pain/ow-2.mp3
pain/hey-stop.mp3
pain/that-hurt.mp3
```
